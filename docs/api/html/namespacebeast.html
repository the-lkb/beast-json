Value beast::parse_strict ( Document & doc,
std::string_view json )

Parse json into doc with strict RFC 8259 compliance. Rejects: trailing commas, leading zeros, invalid escapes, unescaped control characters, trailing content, etc. Throws std::runtime_error describing the violation and its byte offset.
