Detailed Description

The primary accessor type for the Beast JSON DOM.

Value is a lightweight handle consisting of a pointer to the owning DocumentView and a 32-bit tape index. It provides zero-copy, on-demand access to parsed JSON nodes. An invalid (null) Value{} is returned by any access that fails, such as a missing key or out-of-range index.
