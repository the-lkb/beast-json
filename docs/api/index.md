## Core APIs

- **`beast::json::parse`**: The entry point for parsing strings or buffers into a `Document`.
- **`beast::json::dump`**: Unified serialization for scalar types, containers, and custom structs.
- **`beast::json::Value`**: The primary interaction handle for the Tape DOM.
- **`BEAST_JSON_FIELDS`**: The declarative macro for zero-boilerplate object mapping.
