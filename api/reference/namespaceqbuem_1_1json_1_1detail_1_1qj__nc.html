Convert double number from binary to decimal. The output significand is shortest decimal but may have trailing zeros.

This function use the Schubfach algorithm: Raffaello Giulietti, The Schubfach way to render doubles, 2020. https://drive.google.com/open?id=1luHhyQF9zKlM8yJ1nebU0OgVYhfC6CBN https://github.com/abolz/Drachennest

See also: Dragonbox: A New Floating-Point Binary-to-Decimal Conversion Algorithm, 2020. https://github.com/jk-jeon/dragonbox/blob/master/other_files/Dragonbox.pdf https://github.com/jk-jeon/dragonbox

Parameters
sig_rawThe raw value of significand in IEEE 754 format.
exp_rawThe raw value of exponent in IEEE 754 format.
sig_binThe decoded value of significand in binary.
exp_binThe decoded value of exponent in binary.
sig_decThe output value of significand in decimal.
exp_decThe output value of exponent in decimal.
Warning
The input double number should not be 0, inf, nan.
